## 🎬 Cinematic Collage Webapp

A **Spider-Verse** inspired cinematic collage webapp -- a romantic gift featuring photos, videos, procedurally-generated jazz music, and poetic captions.

### Features
- **Spider-Verse Cinematic Style** -- halftone overlays, chromatic aberration, glitch effects, comic-panel layouts, Ben-Day dot cursor trails
- **Gallery, Timeline & Favorites** -- three views to showcase your moments
- **Admin Dashboard** -- password-protected upload & media management (default password: `love`)
- **Procedural Jazz Music** -- original jazz generated in real-time via Web Audio API (no copyright issues)
- **Romantic Captions** -- inspired by Jaden Smith's romantic music style
- **Standalone** -- pure HTML/CSS/JS, no build tools, just open `index.html`
- **LocalStorage** -- all data stored in the browser

### Quick Start
1. Open `index.html` in a browser
2. Click "Enter Our Universe"
3. Navigate to Admin > enter password `love` > upload photos/videos
4. Share the folder with your special person
